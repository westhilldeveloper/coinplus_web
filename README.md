A Next.js starter project (App Router) with Tailwind CSS. This is a minimal scaffold for a chit-fund style website.

## Setup

1. Install dependencies
```bash
npm install
```

2. Run development server
```bash
npm run dev
```

3. Open http://localhost:3000

## Notes

- Replace placeholder images in `public/images`.
- Customize branding variables in `styles/globals.css`.
- This repo includes example components: Header, Hero, PlansGrid, PlanCard, WhyUs, Footer.
